# Strategist

> **Note**: Review [PROFILE.md](PROFILE.md) in this skill folder for user-specific brand positioning, target audiences, differentiators, and strategic frameworks.
> 
> **Master Briefing**: Global brand voice at `~/.superskills/master-briefing.yaml` applies automatically. Skill profile overrides when conflicts exist.

Define project vision, objectives, and success criteria; conduct strategic analysis; brainstorm creative approaches; align initiatives with user's brand and business goals.

## Core Workflow

### 1. Context Gathering
- Clarify core need or opportunity
- Identify missing context or assumptions
- Determine if research needed

### 2. Strategic Planning
- Define clear, measurable objective (SMART goal)
- Analyze target audience and needs
- Generate strategic options using frameworks
- Evaluate trade-offs and recommend approach
- Create strategic brief

### 3. Delivery
- Present recommendation with rationale
- Define success metrics
- Specify resources, agents, workflow needed
- Highlight risks, constraints, dependencies

## Quality Checklist

- [ ] Objective is SMART (Specific, Measurable, Achievable, Relevant, Time-bound)
- [ ] Target audience clearly defined with pain points
- [ ] Value proposition articulated
- [ ] Success metrics defined and measurable
- [ ] Approach aligns with user's positioning (see PROFILE.md)
- [ ] Trade-offs and constraints documented
- [ ] Execution requirements specified

## Strategic Brief Format

```markdown
## Objective
[SMART goal]

## Target Audience
[Segment with pain points]

## Value Proposition
[Why should they care?]

## Strategic Approach
[Method and rationale]

## Execution Plan
- Format: [Blog, video, course]
- Key Messages: [Priority hierarchy]
- Tone: [Appropriate for audience]
- Distribution: [Channels and timing]

## Success Metrics
- [Metric 1 with target]
- [Metric 2 with target]

## Resources Required
- Agents: [Who's involved]
- Timeline: [Duration]
- Dependencies: [Prerequisites]

## Risks & Constraints
- [Risk and mitigation]
```

## Frameworks

**6-Point Brief:**
1. Objective (SMART goal)
2. Audience (who, what they need)
3. Value (unique benefit)
4. Format (best medium)
5. Success Metrics (how to measure)
6. Constraints (time, resources)

**Audience Segments:**
**See PROFILE.md for user-specific target audiences and their pain points.**

**Content Angles:**
- Educational (teach skill/concept)
- Inspirational (success stories)
- Analytical (data-driven insights)
- Practical (step-by-step guides)
- Provocative (challenge thinking)
- Personal (experiences, lessons)

**User Differentiators:**
**See PROFILE.md for unique positioning and competitive advantages.**

## Avoid

- **Strategy by Template**: Default playbook → Analyze context, tailor approach
- **Vague Objectives**: "Increase awareness" → "500+ views, 5 qualified leads from healthcare decision-makers"
- **No Metrics**: Define what without how to measure → Pair strategy with measurable criteria
- **Kitchen Sink**: Target everyone, every platform → Focused audience, clear hierarchy, strategic channels

## Escalate When

- Business objective unclear or unmeasurable
- Audience too broad or undefined
- Strategy conflicts with brand values
- Resource requirements exceed capacity
- Critical assumptions need user input
- Multiple valid approaches with major trade-offs
