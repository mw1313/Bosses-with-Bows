## Prompt Engineering Best Practices

### Effective Prompt Structure
```
[Task] + [Context] + [Style/Format] + [Constraints]

Example:
"Write a project proposal for implementing AI tools in our 
marketing department. Include sections on goals, timeline, 
budget, and expected ROI. Use professional but accessible 
language. Keep it to 2 pages."
```

### The 5 W's for Better Prompts
- **What**: Specific task or output needed
- **Who**: Audience or perspective
- **Where**: Context or location
- **When**: Timeframe or urgency
- **Why**: Purpose or goal

### Prompt Templates by App

**Gmail - Email Composition:**
```
"Draft an email to [recipient] about [topic]. 
Tone: [professional/casual/formal]. 
Include: [key points]. 
Length: [brief/detailed]."
```

**Docs - Content Creation:**
```
"Create a [document type] about [topic] for [audience]. 
Include sections on [points]. 
Style: [tone and voice]. 
Length: [word count/pages]."
```

**Sheets - Data Analysis:**
```
"Analyze this data to [goal]. 
Create [chart/pivot table] showing [insight]. 
Highlight [metrics] and provide [number] recommendations."
```

**Slides - Presentation:**
```
"Create a [presentation type] about [topic] for [audience]. 
[Number] slides covering [key points]. 
Style: [minimal/creative/professional]. 
Include images for each slide."
```

**Meet - Meeting Prep:**
```
"Summarize this document and create talking points for 
a meeting about [topic]. Include questions to ask and 
potential objections to address."
```

## Gemini vs. Copilot Comparison

### When to Choose Gemini
- **Primary Workspace**: Already using Google Workspace
- **Collaboration focus**: Real-time team collaboration critical
- **Cost**: More affordable pricing
- **Simplicity**: Easier setup and management
- **Mobile**: Better mobile app integration

### When to Choose Copilot
- **Primary Workspace**: Already using Microsoft 365
- **Enterprise features**: Advanced compliance needs
- **Depth**: More sophisticated in Office apps
- **Integration**: Deeper integration with Windows ecosystem

### Feature Comparison

| Capability | Gemini | Copilot |
|------------|--------|---------|
| Email drafting | ✓ Good | ✓ Good |
| Document creation | ✓ Good | ✓ Excellent |
| Data analysis | ✓ Good | ✓ Excellent |
| Presentation design | ✓ Excellent (images) | ✓ Good |
| Meeting summaries | ✓ Good | ✓ Excellent |
| Cross-app intelligence | ✓ Developing | ✓ Strong |
| Pricing | Lower | Higher |
| Availability | Workspace plans | M365 E3/E5 |

## Advanced Techniques

### Iterative Prompting
Start broad, refine with follow-ups:
1. Initial prompt: General request
2. Review output, identify gaps
3. Follow-up: "Add more detail on..."
4. Follow-up: "Change tone to..."
5. Final: "Format as..."

### Context Stacking
Build on previous interactions:
- "Using the outline we just created..."
- "Based on the analysis above..."
- "Following the same format..."

### Constraint Optimization
Be specific about limits:
- Word/page count
- Tone and style
- Required sections
- Data sources to use
- Format requirements

## Avoid

- **Vague Prompts**: "Write something" → Specific task with details
- **One-Shot Only**: Single prompt → Iterate and refine
- **No Review**: Trust blindly → Always verify and edit
- **Ignore Context**: Generic prompts → Include relevant background
- **Over-Reliance**: Let AI do everything → Use as augmentation
- **No Documentation**: Lose good prompts → Build prompt library

## Escalate When

- Data privacy or compliance concerns
- Need for custom integration (Apps Script)
- Enterprise deployment questions
- Technical issues with Gemini access
- Comparison with other tools needed
- ROI not meeting expectations
- Training needs beyond individual use
